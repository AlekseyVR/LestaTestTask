# RPG Game Project with tests from Lesta

## Описание проекта
### Проект является тестовым заданием на позицию QA Automation.

Проект представляет собой RPG-игру, разработанную на Python.
А также тесты по стратегии white-box.
Основной файл игры расположен по пути `root\rpg_game\main.py`.

## Установка зависимостей
1. Убедитесь, что у вас установлен Python версии 3.10
2. Установите необходимые зависимости командой:
   ```bash
   pip install -r requirements.txt
   ```

## Запуск игры
Для запуска основной программы выполните:
```bash
python rpg_game\main.py
```

## Запуск тестов
Для выполнения тестов используйте команду:
```bash
python -m pytest tests/
```

## Структура проекта
- `main.py` - основной исполняемый файл игры
- `tests/` - директория с тестами
- `requirements.txt` - файл с зависимостями проекта
